### 编译运行

当在gradle.properties中设置IsBuildMudle=true时，可以独立运行每个module，独立运行调试。
当设置IsBuildMudle=false，可以编译运行整个project，注意IsBuildMudle变量设置改变时，要对gradle进行sysn。
